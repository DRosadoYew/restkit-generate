# restkit-generate

A model generator for iOS RestKit

## Usage

`./restkit-generate.rb model Business \
address:NSString \
bearing:NSNumber \
city:NSString \
created_at:NSDate \
description:NSString \
distance:NSNumber \
email:NSString \
enabled:NSNumber \
has_age_restriction:NSNumber \
has_drinks:NSNumber \
has_food:NSNumber \
id:NSNumber \
latitude:NSNumber \
longitude:NSNumber \
name:NSString \
phone:NSString \
photo_file_name:NSString \
state:NSString \
updated_at:NSDate \
website:NSString \
zip:NSString`
